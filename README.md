# 🐕 Petio Dog Daycare Management System

A Python-based object-oriented daycare management system for tracking dogs, scheduling activities, and managing daily operations.

![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## Features

- **Dog Management**: Track energy, hunger, happiness, and special needs
- **Daily Operations**: Check-in/out, group playtime, feeding schedules, nap time
- **Revenue Tracking**: Automated billing per dog per day
- **Smart Logic**: Conditional activities based on dog stats (tired dogs auto-nap)
- **Special Needs**: Support for dietary restrictions and medical requirements
